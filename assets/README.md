# Portfolio Assets Guide

Add your images to this folder. The portfolio expects the following files:

## Required Images

- **port.jpg** — Your profile/portrait photograph (recommended: 400×400px, square format)
- **project-1.jpg** — "Adaptive Learning Platform" project thumbnail (recommended: 480×300px)
- **project-2.jpg** — "Human-AI Collaboration Study" project thumbnail (recommended: 480×300px)
- **project-3.jpg** — "Open Data Initiative" project thumbnail (recommended: 480×300px)

## Optional Files

- **CV.pdf** — Your Curriculum Vitae (PDF format, for download link)

## Image Tips

- Use high-quality images (at least 72 DPI for print-ready)
- Compress images to keep page load time fast (use tools like TinyPNG or ImageOptim)
- Portrait should be professional (headshot against neutral background)
- Project thumbnails should be representative of the work (screenshot, diagram, or cover image)

## Customization

To add or change image filenames, edit the `src` attributes in `home.html` or update references in this guide.
